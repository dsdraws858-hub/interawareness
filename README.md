# 🌍 InterAwareness

Learn the issues. Understand the impact. Inspire change.

InterAwareness is a global awareness platform dedicated to educating people about climate change, human rights, inequality, emerging technologies, and other pressing world issues.

## Features

- 🌎 Interactive Issue Pages
- 💬 Community Discussions
- ❓ Searchable FAQ
- 🌙 Dark/Light Mode
- 📊 Statistics and Visualizations
- 📚 Learning Resources

## Mission

To foster empathy, critical thinking, informed dialogue, and evidence-based learning in a hopeful, community-driven environment.
